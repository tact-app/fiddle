## üõ¨ Installation

### Homebrew

```bash
$ brew install octolab/tap/fiddle
```

For more details please visit [the page](https://homebrew.octolab.org/).

### Binary

```bash
$ curl -fsSL https://install.octolab.org/fiddle | sh
# or
$ wget -qO-  https://install.octolab.org/fiddle | sh
```

For more details please visit [the page](https://install.octolab.org/).

### Source

```bash
# use standard go tools
$ go get go.octolab.org/toolset/fiddle@latest
# or use egg tool
$ egg tools add go.octolab.org/toolset/fiddle@latest
```

For more details please visit [the page](https://egg.octolab.org/).

### Shell completions

```bash
$ fiddle completion > /path/to/completions/...
# or
$ source <(fiddle completion)
```
